# calculatorApp
A copied version of the iOS calculator app.

# Install node modules:
Run nmp install before starting the project.

# Run project
After running npm install, run npm start to start the project.

# Simulator
Option 1: Scan the QR code in the terminal to run the app on you mobile EXPO app.
Option 2: Follow the instructions and click "i" to open up your iOS simulator to run the app.
